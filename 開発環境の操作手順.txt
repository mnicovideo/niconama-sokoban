■■■ cloud9ideでよく使うgitのコマンド ■■■

■ git status
  ソースのステータスを確認

■ git pull
  リポジトリで変更のあった更新を作業用プロジェクトに反映させる
 
■ git add . (またはファイル名)
  リポジトリに反映させるファイルを選択。
  .でカレントディレクトリ以下にある変更されたファイルを自動で選択。

■ git commit -m "変更内容"
  選択したファイルの確定。-mオプションで変更内容の概要を記述する。

■ git push
  リポジトリへ変更を反映させる

■■ cd、pwdコマンドについて
  git のコマンドではないが操作場所の選択に使用する。

  ・cd でホームディレクトリに移動
  ・cd ディレクトリ名 でディレクトリに移動
  ・cd .. で一つ上のディレクトリ階層へ移動
